## 📖 About this application

Gitmap-v16 helps you see the structure of your software projects. Projects grow over time as you add folders and files. This application maps those files into a visual format. You see how your project connects. It creates a simple chart of your code base. You move through your folders and see the architecture of your work. This tool removes the need to click through hundreds of directories.

## 💻 System requirements

Your computer needs Windows 10 or Windows 11 to run this software. You need at least 4GB of RAM to process projects of a standard size. Keep 100MB of free disk space available for the installation files. Gitmap-v16 works best with a modern processor. We recommend an Intel Core i5 or an AMD Ryzen 5 for smooth performance. Ensure you have the latest Windows updates installed before you start the setup process.

## 🚀 Using Gitmap-v16

1. Find the Gitmap-v16 icon on your desktop or in your Start menu.
2. Click the icon to open the main window.
3. Select File in the top menu bar.
4. Choose Open Project Folder from the dropdown list.
5. Search for the folder on your computer that contains your project code.
6. Select that folder and click OK.
7. The application scans your files. This takes a few seconds for small projects.
8. A map appears in the center of the window. 
9. Use your mouse to click on specific lines to see more detail. 
10. Use the zoom slider at the bottom to adjust the view.

## 🛠️ Viewing your project maps

The map shows the root folder of your project at the top level. Each subfolder appears as a branch connected to the root. Files appear as small nodes at the end of these branches. Color codes help you identify file types. Blue nodes represent code files. Grey nodes show text or document files. Red nodes highlight configuration files. Click any node to open a side panel. This panel lists the file path and the file size.

## 🔍 Searching for files

If your project contains many files, use the search box at the top right of the screen. Type the name or part of the name of the file you seek. The map dims all other parts of the project. It highlights the match and its exact location. Press the clear button to reset the view to the full project map.

## 📁 Exporting your data

You often need to share your project structure with your team. Select Export from the File menu to generate a summary. Choose between PDF or Image format for your export. The software saves the file to your documents folder. You can now email or print this map for your team meetings.

Your settings stay the same during this process. Do not uninstall the older version unless the installation instructions tell you to do so. 

## ❓ Troubleshooting common issues

If the application does not open, check if your antivirus software placed a block on the program. Some security tools flag new software for safety checks. Add Gitmap-v16 to your trusted list if this happens. If the map window stays blank, verify that the folder you selected contains actual files. Gitmap-v16 ignores empty or hidden system folders to keep the map clean. Reset the view by closing and reopening the project folder if the map looks distorted after a screen resize. Use the View menu to reset the window frames.
